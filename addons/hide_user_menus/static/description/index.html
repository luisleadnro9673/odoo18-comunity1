App Overview

The Hide User Menus app is designed to simplify your Odoo interface by removing unnecessary menu items from the top right corner. By hiding elements like Documentation, Support, Shortcuts, this app helps create a more focused and clutter-free user experience.

Key Features:

  • Hides Documentation, Support, Shortcuts, Onboarding, and My Odoo.com account from the top-right user menu
  • Streamlines the Odoo interface for a cleaner user experience
  • Reduces UI clutter, making it easier for users to focus on important tasks
